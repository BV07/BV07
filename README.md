## Featured Projects

### 📰 [Market Analysis for Maine Trust for Local News (Fall 2025)](https://github.com/BV07/DS5110-FinalProject-METLN)
Exploratory data analysis identifying underserved opportunity markets for community newspapers across Maine. Built a prototype dashboard (not public) for visualizing engagement metrics and demographic patterns.

**Tech:** Python, Pandas, PowerBI, data visualization  
**Outcome:** Identified York County as key expansion opportunity based on penetration rate analysis

---

### 🌌 Galaxy Spectral Analysis Training (Summer 2025) *(coming to GitHub soon)*
Interactive Jupyter notebooks for processing and analyzing spectral data from SDSS surveys (MaNGA & eBOSS). Used to study gas outflows in millions of galaxies across cosmic time.

**Tech:** Python, Astropy, SciPy, Matplotlib, Pandas  

---

### 🔬 nEXO Signal Denoiser (Summer 2024) *(coming to GitHub soon)*
Deep neural network for denoising time-series waveforms in nuclear physics experiment (nEXO collaboration). Built supervised denoising architecture with CNN-based encoder, trained on 40,000 synthetic waveform pairs generated via Monte Carlo methods.

**Tech:** TensorFlow, Keras, NumPy, Monte Carlo simulation  
**Impact:** Proposed for integration into UMass Amherst lab's data analysis pipeline
